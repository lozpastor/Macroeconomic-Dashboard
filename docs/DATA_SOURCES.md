# Fuentes De Datos

## Principios

- Usar fuentes oficiales o reputadas.
- Guardar proveedor, frecuencia y vintage de cada observacion.
- No mezclar metodologias sin normalizacion explicita.
- Mantener trazabilidad por indicador y pais.

## Mapeo inicial

| Categoria | Indicador | Fuente prioritaria |
| --- | --- | --- |
| PIB | GDP Current USD, Constant USD, PPP, per capita, growth | World Bank, IMF |
| Inflacion | CPI, Core CPI, YoY, MoM | World Bank, IMF, OECD |
| Laboral | Unemployment, participation, employment growth | World Bank, OECD |
| Exterior | Current account, trade balance, exports, imports, FX reserves | World Bank, IMF |
| Fiscal | Debt, Debt/GDP, fiscal balance, expenditure, tax revenue | IMF, World Bank |
| Monetaria | Interest rate, central bank rate, M1, M2 | ECB, FRED, IMF |
| Financiero | Credit growth, bank assets, NPL | BIS, IMF |
| Demografia | Population, growth, urbanization | World Bank, UN |
| Desarrollo | HDI, poverty, education | UNDP, World Bank |
| Energia | Consumption, renewables, CO2 | World Bank, IEA cuando haya licencia |

## Frecuencia de actualizacion

- Alta frecuencia: cada hora si proveedor y licencia lo permiten.
- World Bank/UN/OECD: diaria o semanal, segun disponibilidad.
- IMF/BIS/ECB/FRED: diaria para series monetarias y financieras.
